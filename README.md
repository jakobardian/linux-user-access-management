# Linux User Access Management Lab


## Objectives
Repository ini berisi proses pembelajaran dan praktik dasar Linux
user, group, permission dan sudo acces management.


## Scope
- User and group management
- File and directory permissions
- Ownership management
- Sudo configuration and privilege control


## Environment
- VMware Workstation Pro 25H2
- Ubuntu Desktop 24.04 LTS


## Tools & Commands
- useradd, usermod, userdel
- groupadd, gpasswd
- chmod, chown, umask
- sudo, visudo


## Learning Approach
- Hands-on dengan command step by step
- Skenario kasus/insiden
- Dokumentasi error dan personal insight


## Repository Structure
- labs/     : Command step by step berdasarkan praktik nyata
- notes.md  : Mistakes,observation dan lesson learned


## Status
**Learning in progress**
